Downloads

Todas as versões.

100% gratuito e open source. Confira o SHA-256 após baixar (shasum -a 256 arquivo no macOS · certutil -hashfile arquivo SHA256 no Windows). Builds sem notarização Apple: na 1ª abertura use Ajustes do Sistema → Privacidade e Segurança → "Abrir Assim Mesmo"; se o macOS disser "danificado" (versões ≤ 0.2.12 ou download antigo), rode xattr -cr /Applications/HyperFrame.app no Terminal. Windows/SmartScreen → "Mais informações" → "Executar assim mesmo". Linux: compile do fonte (npm i && npm run tauri build).
